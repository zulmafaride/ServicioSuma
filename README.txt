PASOS
1) se debe ejecutar el archivo start.sh
2) Luego desde el navegador se debe ingresar a la direccion http://localhost:8080/ServicioRestSuma/app/suma?numero1=300&numero2=20,
   si se quiere cabiar los valores de la suma se debe modificar los valores de los parametros numero1 y numero2
3) para terminar el servicio se debe ejecutar el archivo stop.sh
